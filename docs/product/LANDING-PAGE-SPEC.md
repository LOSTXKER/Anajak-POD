# 📄 Landing Page Specification
## Anajak POD - Print on Demand Platform

---

## 🎯 Overview

Landing Page สำหรับ Anajak POD ที่ออกแบบมาเพื่อ:
- อธิบายให้คนไทยเข้าใจว่า **POD (Print on Demand) คืออะไร**
- รองรับ **2 โหมดการใช้งาน** (ทำขาย vs ซื้อใส่เอง)
- สามารถ **ออกแบบสินค้าได้เลยโดยไม่ต้อง Login**
- **เชื่อมโยงกับเว็บโรงงาน Anajak T-Shirt ที่มีอยู่แล้ว**

## 🚀 Core USP: "ทำเสื้อได้ภายใน 5 นาที"

### Key Messages

| Type | Message |
|------|---------|
| **Headline หลัก** | "ทำเสื้อได้ภายใน 5 นาที" |
| **Subheadline** | "ไม่ต้องคุยกับโรงงาน ไม่ต้องรอใบเสนอราคา ไม่ต้องแก้ไฟล์" |
| **Supporting** | "ออกแบบเอง ระบบเช็คให้ สั่งได้ทันที" |
| **Comparison** | "สั่งทำเสื้อแบบเดิมใช้เวลา 2-3 สัปดาห์ กับ Anajak POD ใช้เวลาแค่ 5 นาที" |

### Before vs After Comparison

| แบบเดิม 😫 | Anajak POD 🚀 |
|-----------|--------------|
| ติดต่อโรงงาน 1-2 วัน | เปิดเว็บออกแบบเลย 0 นาที |
| คุยเรื่องแบบ/ไฟล์ 2-5 วัน | ระบบเช็คไฟล์อัตโนมัติ ทันที |
| รอใบเสนอราคา 1-2 วัน | เห็นราคาทันที |
| ตกลง + โอนเงิน 1 วัน | ชำระออนไลน์ 2 นาที |
| ผลิต + ส่ง 5-10 วัน | ผลิต + ส่ง 2-3 วัน |
| **รวม: 10-20 วัน** 🐢 | **รวม: 2-3 วัน** ⚡ |

### 6 เหตุผลที่เราเร็วกว่า

1. ❌ ไม่ต้องคุยกับใคร → Self-service
2. ❌ ไม่ต้องส่งไฟล์ไปมา → อัพโหลดในระบบ
3. ❌ ไม่ต้องรอใบเสนอราคา → เห็นราคาทันที
4. ❌ ไม่ต้องแก้ไฟล์หลายรอบ → ระบบบอกปัญหาทันที
5. ✅ ไฟล์พร้อมผลิตทันที → เข้าคิวผลิตได้เลย
6. ✅ เราเป็นโรงงานเอง → ไม่มี Middleman

### Taglines ที่ใช้ได้

- "ทำเสื้อได้ภายใน 5 นาที"
- "5 นาทีออกแบบ 3 วันถึงมือ"
- "ไม่ต้องคุยกับใคร ออกแบบเองได้เลย"
- "บอกลาการรอ 2 อาทิตย์"
- "Design in 5, Deliver in 3"

## 🔀 User Segments

### เปรียบเทียบ 3 กลุ่มเป้าหมาย

| หัวข้อ | 🛒 ทำขาย | 👕 ซื้อใส่เอง | 🏢 องค์กร |
|--------|---------|-------------|----------|
| **Target** | ผู้ประกอบการ, Influencer | คนทั่วไป, กลุ่มเพื่อน | HR, ฝ่ายจัดซื้อ |
| **จำนวนสั่ง** | ไม่จำกัด (ขายต่อ) | 1-10 ชิ้น | 20-500+ ชิ้น |
| **ราคา** | ต้นทุน + กำไร | ราคาปกติ | ส่วนลด Bulk 10-25% |
| **เอกสาร** | Dashboard รายงาน | ใบเสร็จธรรมดา | ใบกำกับภาษี |
| **อยู่ในหน้า** | Landing (โหมด A) | Landing (โหมด B) | หน้าแยก /corporate |

### โหมด A: "ทำขาย" (Seller Mode) 🛒

**Key Messages:**
- สร้างแบรนด์ของตัวเองได้ง่าย ไม่ต้องสต็อกสินค้า
- ต้นทุนต่ำ กำไรสูง (40-60%)
- เชื่อมต่อร้านค้าออนไลน์ได้ทันที
- ระบบจัดส่งอัตโนมัติ (Fulfillment)

### โหมด B: "ซื้อใส่เอง" (Personal Mode) 👕

**Key Messages:**
- สั่งได้ตั้งแต่ 1 ชิ้น ไม่มีขั้นต่ำ
- คุณภาพพรีเมียม เกรดส่งออก
- รองรับงานกลุ่ม ทุกโอกาส
- ส่งถึงมือใน 3-5 วัน

### Use Cases แบ่งตามขนาด

**🔹 Small (1-10 ชิ้น):** ของขวัญวันเกิด, เสื้อคู่รัก, เสื้อครอบครัว

**🔸 Medium (10-30 ชิ้น):** เสื้อแก๊ง, เสื้อทีมกีฬา, งานปาร์ตี้

**🔶 Large (30-100+ ชิ้น):** เสื้อรุ่น, งานบวช, งานแต่ง → ลิงก์ไป Use Case Pages

### 5. How It Works

#### สำหรับ Seller:
1. 🎨 **ออกแบบสินค้า** → ใช้เครื่องมือออกแบบฟรี
2. 🏬 **ตั้งราคาขาย** → กำหนดกำไรที่ต้องการ
3. 📢 **โปรโมทร้าน** → แชร์ไป Social / เชื่อมต่อ Marketplace
4. 💰 **รับเงินเมื่อขาย** → เราผลิตและจัดส่ง คุณรับกำไร

#### สำหรับ Personal:
1. 🔍 **เลือกสินค้า** → เลือกจากแคตตาล็อก
2. 🎨 **ออกแบบ** → อัพโหลดรูป/ข้อความ หรือใช้ Template
3. 🛒 **สั่งซื้อ** → ใส่ที่อยู่จัดส่ง ชำระเงิน
4. 📦 **รอรับของ** → จัดส่งถึงบ้าน 3-5 วัน

## 📑 Additional Pages (Brief Spec)

### `/corporate` - B2B Landing Page

**Target:** HR, ฝ่ายจัดซื้อ, เจ้าของธุรกิจ SME

**Key Sections:**
1. Hero: ข้อเสนอสำหรับองค์กร
2. B2B Benefits (ส่วนลด, ใบกำกับภาษี, Account Manager)
3. Bulk Pricing Table
4. Use Cases Gallery (ยูนิฟอร์ม, อีเวนต์, Team Building)
5. Testimonials จากองค์กร
6. Process Timeline
7. Lead Capture Form
8. Contact Sales Section

### `/use-cases/[slug]` - Template Structure

แต่ละหน้าจะมี:
- Hero เฉพาะ Use Case
- Gallery ตัวอย่างงานจริง
- ราคาและส่วนลดสำหรับจำนวน
- Template สำเร็จรูป
- รีวิวจากลูกค้า
- FAQ เฉพาะ Use Case
- CTA: ออกแบบเลย / ขอใบเสนอราคา
