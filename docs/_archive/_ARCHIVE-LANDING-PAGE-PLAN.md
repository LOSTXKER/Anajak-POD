# 📌 Landing Page Planning Document
## Anajak POD - Print on Demand Platform

---

## 🎯 Overview

Landing Page สำหรับ Anajak POD ที่ออกแบบมาเพื่อ:
- อธิบายให้คนไทยเข้าใจว่า **POD (Print on Demand) คืออะไร**
- รองรับ **2 โหมดการใช้งาน** ที่ต่างกัน (ทำขาย vs ซื้อใส่เอง)
- ดึงดูดให้ผู้เข้าชมลองใช้บริการ
- สามารถ**ออกแบบสินค้าได้เลยโดยไม่ต้อง Login**
- เป็นมิตรกับ SEO
- **เชื่อมโยงกับเว็บโรงงาน Anajak T-Shirt ที่มีอยู่แล้ว**

## 🔗 Brand Architecture: Anajak T-Shirt × Anajak POD

### 🏭 สถานะปัจจุบัน

| เว็บไซต์ | ลักษณะ | Target Audience |
|---------|--------|-----------------|
| **Anajak T-Shirt** | เว็บโรงงาน (มีอยู่แล้ว) | B2B, องค์กร, ขายส่ง |
| **Anajak POD** | Platform POD (ใหม่) | ผู้ขายออนไลน์, คนทั่วไป |

### ✅ ข้อดีของการเชื่อมโยง

1. **Trust Transfer** - ความน่าเชื่อถือของโรงงานส่งต่อมา POD
2. **SEO Boost** - Backlink จากเว็บที่มี Authority
3. **Proof of Quality** - "ผลิตจากโรงงานจริง ไม่ใช่ Dropship จากจีน"
4. **Unique Selling Point** - "เราเป็นโรงงานเอง ไม่ใช่ตัวแทน"

### 🔑 เหตุผลที่เราเร็วกว่า

```
1. ❌ ไม่ต้องคุยกับใคร
   → Self-service ออกแบบเองได้เลย

2. ❌ ไม่ต้องส่งไฟล์ไปมา
   → อัพโหลดในระบบ เช็คคุณภาพอัตโนมัติ

3. ❌ ไม่ต้องรอใบเสนอราคา
   → เห็นราคาทันทีตอนออกแบบ

4. ❌ ไม่ต้องแก้ไฟล์หลายรอบ
   → ระบบบอกปัญหาทันที (DPI ต่ำ, ขนาดไม่พอ)

5. ✅ ไฟล์พร้อมผลิตทันที
   → เข้าคิวผลิตได้เลยหลังชำระเงิน

6. ✅ เราเป็นโรงงานเอง
   → ไม่มี Middleman, ผลิตได้ทันที
```
